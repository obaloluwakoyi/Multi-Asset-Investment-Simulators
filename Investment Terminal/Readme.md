Quantum Analytics Suite: Equity & Portfolio Terminals
The Quantum Analytics Suite is a high-end collection of client-side financial engines designed for sophisticated equity portfolio management and executive reporting. These tools automate capital allocation, dividend liquidity forecasting, and multi-year ROI projections using real-time data visualization.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Modern Dark-Mode UI)

Data Visualization: Chart.js

Document Generation: html2pdf.js

📊 Strategic Equity Tools
1. Quantum Strategic Terminal (ngx_portfolio_builder.html)
A robust portfolio construction engine pre-configured for Nigerian Stock Exchange (NGX) equities and global technology assets.

Algorithmic Allocation: Features a "Quick-Pick" selection logic to automate buy units based on total investment capital.

Dividend & Growth Engine: Projects 5-year capital appreciation alongside annual dividend liquidity forecasts.

Asset Management: Tracks ticker-specific metrics including price-per-unit, dividend yields, and targeted growth percentages.

2. Executive KPI Dashboard (executive_kpi_dashboard.html)
An elite, reporting-centric variant of the Quantum engine optimized for high-stakes financial summaries and stakeholder presentations.

Stakeholder Reporting: Specifically engineered for "Executive KPI" reporting with clean, minimalist aesthetics.

Automated PDF Generation: Features a high-fidelity export tool to convert dynamic financial dashboards into print-ready PDF memos.

Metric Synthesis: Distills complex portfolio data into four critical KPIs: Total Projected Wealth, Annual Cashflow, Total Volume, and ROI percentage.

🚀 Key Functionalities
Zero-Server Architecture: Fully functional client-side logic—no database or backend required.

Predictive Modeling: Interactive 5-year growth charts with conservative adjustment factors (90% growth modeling) to ensure realistic projections.

Responsive Design: Grid-based layouts optimized for large-scale financial monitoring and analysis.

📂 Deployment
Simply open either HTML file in a modern web browser to begin simulating portfolio strategies. For professional distribution, use the Export to PDF function to generate formatted investment summaries.
